房貸試算程式簡短報告
1. 使用說明書

本程式為「房貸試算器」，主要功能為計算每月應繳金額、累積利息、剩餘本金，並支援寬限期及深色/淺色模式切換。

使用步驟：

輸入房貸資料：房價、頭期款、年利率、貸款年數、寬限期
使用 TrackBar 調整利率與年限，滑桿同步更新欄位及 Label
點擊「計算」或按 Enter 鍵計算房貸，結果顯示於 Label 與 DataGridView
點擊「匯出」可存成 CSV 檔
點擊「主題切換」切換深色/淺色模式

建議：可在 Word 插入程式介面截圖，標註欄位及按鈕功能

2. 問題排除與學習心得

開發過程遇到的挑戰：

寬限期計算：分為寬限期與正式還款期計算利息與本金
浮點數累積誤差：使用 Math.Max(balance,0) 修正
自動更新計算：綁定 TextChanged 事件
GUI 設計與可用性：DataGridView 清楚呈現期數、本金、利息、剩餘本金；加入深色/淺色模式

學習心得：熟悉 Windows Forms 控制元件及事件綁定，練習浮點數計算與等額本息公式應用，除錯技巧，Github 操作。

3. 公式與資料來源

房貸計算公式（等額本息，含寬限期）：

月利率: r = 年利率 / 12
貸款本金: P = 房價 × (1 - 頭期款%)
正式還款期月付金額: M = P * r*(1+r)^n / ((1+r)^n -1), n = 總期數 - 寬限期月數
首期利息: 利息 = 剩餘本金 × 月利率
首期本金: 本金 = 月付金額 - 利息
